# 🌐 Portfolio Website – Devant Pal

A modern, responsive personal portfolio website showcasing my skills, projects, and experience as a Cloud & Backend Engineer.

---

## 🚀 Features

- 🔥 Interactive UI/UX design
- 📩 Contact form with EmailJS integration
- 🤖 Auto-reply system for user acknowledgment
- ✅ Form validation
- 🌐 Fully responsive design
- ⚡ Fast and lightweight (no backend required)

---

## 🛠️ Tech Stack

- HTML5
- CSS3
- JavaScript
- EmailJS (Serverless Email Handling)

---

## 📬 Contact Form System

- Sends real-time emails using EmailJS
- Automatically replies to users
- No backend required

---

## 🧠 How It Works

1. User fills contact form
2. JavaScript captures input
3. EmailJS sends email via Gmail API
4. Auto-reply is triggered
5. User receives acknowledgment

## 📂 Project Structure (Simplified)

- `index.html` → Main portfolio file

## ⚙️ Setup Instructions

1. Clone the repository
2. Open `index.html`
3. Add your EmailJS credentials:
   - Service ID
   - Template ID
   - Public Key
